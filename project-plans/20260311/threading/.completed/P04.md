### Tests Added
Added 4 new tests to `rust/src/threading/tests.rs`:

1. **test_thread_c_int_return_positive** (line 88)
   - Spawns `Thread<c_int>` returning 42
   - Joins and asserts result is 42
   - Validates positive return values

2. **test_thread_c_int_return_zero** (line 97)
   - Spawns `Thread<c_int>` returning 0
   - Joins and asserts result is 0
   - Validates zero is distinguishable from error (Ok(0) vs Err)

3. **test_thread_c_int_return_negative** (line 106)
   - Spawns `Thread<c_int>` returning -1
   - Joins and asserts result is -1
   - Validates negative return values and sign preservation

4. **test_thread_c_int_return_multiple_values** (line 115)
   - Spawns 8 threads with boundary values: 0, 1, -1, 42, 255, -128, i32::MAX, i32::MIN
   - Joins all threads and verifies each returns its specific value
   - Validates concurrent threads with diverse return values
