## Key Design Decisions
- Used C bridge wrappers for global setters (Rust can't write C globals directly)
- Used single C wrappers for config loading (uio_DirHandle is C-opaque)
- Kept direct C externs for functions with simple signatures
- All externs in one module for discoverability
