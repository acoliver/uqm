### 2. Replacement Logic in process_resource_desc

**Location:** `rust/src/resource/dispatch.rs:112-128`

**Change:**
```rust
// @plan PLAN-20260314-RESOURCE.P06
// @requirement REQ-RES-OWN-009
// Check if replacing an existing entry with loaded heap data
if let Some(old) = self.entries.get(key) {
    unsafe {
        if !old.data.ptr.is_null() {
            if let Some(handler) = self.type_registry.lookup(&old.type_handler_key) {
                if let Some(free_fun) = handler.free_fun {
                    if old.refcount > 0 {
                        log::warn!(
                            "Replacing resource '{}' with outstanding refcount={}",
                            key,
                            old.refcount
                        );
                    }
                    free_fun(old.data.ptr);
                }
            }
        }
    }
}
```

**Behavior:**
- Before inserting new entry, checks if key already exists
- If old entry has loaded heap data (ptr non-null + handler has free_fun), calls free_fun
- Warns if refcount > 0 but still proceeds with cleanup
- Prevents leaking replaced resources

### 3. cleanup_all_entries Method

- Skips value types (no free_fun) and unloaded entries (ptr null)
