### 3. Logic Correctness

#### Path Normalization (PC-04)
```text
01: FUNCTION normalize_virtual_path(base_virtual, input_path) -> PathBuf
02:   IF input_path == "" RETURN base_virtual
03:   LET combined = IF input_path is absolute THEN input_path ELSE base_virtual.join(input_path)
04:   LET result = empty component stack
05:   FOR EACH logical component in combined
06:     IF component == "" OR component == "." THEN continue
07:     IF component == ".."
08:       IF result not empty THEN pop else stay clamped at root
09:     ELSE push component
10:   RETURN rooted path from result
```

**[OK] Algorithm is SOUND**:
- Handles empty path correctly (line 2)
- Absolute path short-circuit (line 3)
- Component filtering (lines 6-9) matches POSIX normalization requirements
- Root clamping prevents over-traversal (line 8)

#### Mount Confinement (PC-04)
```text
14: FUNCTION map_virtual_to_host(mount_root, mount_relative_components) -> Result<PathBuf, errno>
15:   LET host_components = empty component stack
16:   FOR EACH component in mount_relative_components
17:     IF component == ".."
18:       IF host_components empty
19:         -- do not escape above mount physical root
20:         CONTINUE
21:       ELSE pop host_components
22:     ELSE push component
23:   RETURN mount_root.join(host_components)
```

**[OK] Algorithm is SOUND**:
